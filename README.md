# mitawa-qr
QR para redirigir al perfil de Instagram de Mitawa Coffee y Talleres usando GitHub Pages.

Este repositorio contiene una página simple de redirección que apunta al perfil de Instagram de Mitawa Coffee y Talleres.
Al escanear el QR generado con el enlace de GitHub Pages, los usuarios son llevados automáticamente al perfil, evitando problemas de compatibilidad con lectores QR en dispositivos Android (Samsung, Xiaomi, etc.) y iOS.

Estructura:
- index.html: página de redirección al perfil de Instagram.

Instrucciones de uso:
1. Subir index.html al repositorio.
2. Activar GitHub Pages desde la rama principal.
3. Usar el enlace generado por GitHub Pages para crear el QR.
